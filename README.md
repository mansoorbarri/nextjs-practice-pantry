## Pantry 
a parellel food management app built with Next.js, Clerk, PostgreSQL, uploadthing and Tailwind CSS

## Features
- User authentication with Clerk
- CRUD operations for food items
- Search functionality
- Sorting functionality
- Image uploads
- Responsive design
